# GitCamp

## Overview

GitCamp is a practical repository created during the **Git and GitHub Bootcamp** at Chalimbana University. The repository provides participants with hands-on experience in using Git and GitHub for version control, collaboration, and project management.

Through this repository, learners practice essential Git operations and collaborative workflows commonly used in software development teams.

## Learning Objectives

Participants will learn how to:

* Initialize a Git repository.
* Create and manage GitHub repositories.
* Stage, commit, and track changes.
* Push local projects to GitHub.
* Clone repositories on different machines.
* Pull updates from remote repositories.
* Create and merge branches.
* Resolve merge conflicts.
* Collaborate effectively in teams using GitHub.

## Topics Covered

* Introduction to Version Control
* Git Installation and Configuration
* Basic Git Commands
* Working with Branches
* Remote Repositories
* GitHub Collaboration
* Pull Requests and Code Reviews
* Team-Based Development Workflows

## Practical Activities

During the bootcamp, participants will:

1. Create a GitHub repository.
2. Push local files to GitHub.
3. Clone repositories on another computer.
4. Make and commit changes.
5. Push and pull updates.
6. Work collaboratively in groups.

## Target Audience

This repository is intended for students, lecturers, and software developers who wish to develop practical skills in Git and GitHub for individual and collaborative software development projects.
